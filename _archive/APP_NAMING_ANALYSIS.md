# ShiftWell — App Naming Analysis

> **Decision: Rename from "NightShift" to "ShiftWell"**
> Date: 2026-03-14 | Session 6

---

## Why We Renamed from "NightShift"

### 1. Apple Trademark Conflict — CRITICAL
Apple filed a trademark for "Night Shift" (International Classification 009 — computer software). It's their built-in blue-light filter feature on every iPhone since iOS 9.3 (2016). Attempting to register "NightShift" for a health/sleep software app would almost certainly face opposition.

### 2. App Store Crowding — HIGH RISK
Multiple existing apps use the name:
- **Night Shift Therapy** — FDA-cleared positional sleep apnea app (direct category overlap)
- **Nightshift Dark Mode** — Safari extension (macOS)
- **Nightshift Stargazing** — Astronomy app
- **the NIGHT SHIFT** — A game

Any search for "NightShift" on the App Store returns multiple competing results.

### 3. Brand Confusion
Users searching "Night Shift" in settings or the App Store will associate it with Apple's blue-light filter, not a circadian optimization tool.

## Competitive Name Analysis

| Competitor | Name Analysis |
|---|---|
| **Timeshifter** | "Time" + "shifter." Communicates circadian shifting. Market leader at $9.99/mo |
| **Arcashift** | "Circa" (circadian) + "shift." Scientific but approachable. Built by PhD circadian scientists. $4.99/mo |
| **SleepSync** | "Sleep" + "Sync." Clear, functional. Monash University research project |
| **Propeaq** | Abstract/invented name. Pairs with light glasses hardware. Less intuitive |

### Naming Patterns That Work in Health/Sleep Apps
1. **Compound descriptors**: Sleep Cycle, AutoSleep, SleepWatch — immediately communicate function
2. **Short invented words**: Endel, Noisli, Calm — brandable, memorable, 2 syllables
3. **Science + function mashups**: Timeshifter, Arcashift — signal credibility without being clinical
4. **Action-oriented**: BetterSleep — implies improvement
5. **Under 3 syllables** performs best for recall and voice assistant compatibility

## Names Considered (20 Candidates)

### Shift-Focused
| Name | Pros | Cons |
|---|---|---|
| **ShiftWell** ✅ | Clear meaning (shift + wellness), approachable, positive | "Shift" alone is generic in search |
| ShiftSync | Communicates synchronization/alignment | Similar to SleepSync (Monash) |
| OffShift | Implies recovery time, memorable | Could sound like "leaving work" |
| ShiftGuard | Protective connotation | Sounds like a security app |

### Sleep-Focused
| Name | Pros | Cons |
|---|---|---|
| SleepShift | Perfectly descriptive, intuitive | Very literal, less brandable |
| CircaSleep | Scientific + elegant | "Circa" confused with "approximately" |
| RestSync | Clean, functional | Less differentiated |

### Rhythm / Circadian-Focused
| Name | Pros | Cons |
|---|---|---|
| CircaShift | Echoes "circadian + shift" | Too close to Arcashift (legal risk) |
| RhythmRx | Physician credibility via "Rx" | "Rx" may intimidate non-medical users |
| CircaSync | Scientific, clean | Abstract for general audience |

### Creative / Brandable
| Name | Pros | Cons |
|---|---|---|
| Duskwise | Evocative, unique, highly trademarkable | Doesn't immediately say "shift work" |
| Nocturn | Latin root, elegant, memorable | Could feel dark/negative |
| Restora | Implies restoration, warm | Doesn't signal shift work |
| Lumen | Short, light-related | Already used by metabolic health app |
| Drift | Short, evocative of falling asleep | Used by chat/marketing software |

### Medical Credibility
| Name | Pros | Cons |
|---|---|---|
| ShiftRx | "Rx" = doctor-built credibility | May imply regulated medical product |
| SleepScript | Creative, memorable | Could be confused with coding tool |
| MedShift | Clear medical positioning | Sounds like a staffing app |

## Why ShiftWell Won

1. **Clarity** — Instantly communicates "wellness for shift workers" without explanation
2. **ASO** — Ranks for "shift work," "shift worker health," "wellness shift"
3. **Trademark** — Low risk, no obvious conflicts in Class 9 or 44
4. **Memorability** — Two familiar words, easy to spell, easy to say to Siri
5. **Appeal** — Positive and approachable, not clinical or intimidating
6. **Differentiation** — Unlike competitors who lean scientific (Timeshifter, Arcashift), ShiftWell leads with the *benefit*
7. **Subtitle** — Pairs perfectly: "ShiftWell: Circadian Sleep Plans for Shift Workers"
