**dashboard/components/ROICalculator.tsx:**
Interactive calculator. State: headcount (number), avgAnnualSalary (number). Outputs:
```typescript
// ROI formula from ENTERPRISE-OUTCOMES-FRAMEWORK:
const NIOSH_ABSENTEEISM_COST = 3600;         // per incident
const SHRM_TURNOVER_COST_MULTIPLIER = 0.5;   // 50% of annual salary
const ABSENTEEISM_REDUCTION_PCT = 0.15;      // 15% reduction (conservative)
const TURNOVER_REDUCTION_PCT = 0.10;         // 10% reduction (conservative)
const PRESENTEEISM_COST_PER_PERSON = 1967;  // RAND 2016

const reducedAbsenteeismCost = headcount * NIOSH_ABSENTEEISM_COST * ABSENTEEISM_REDUCTION_PCT;
const reducedTurnoverCost = headcount * avgAnnualSalary * SHRM_TURNOVER_COST_MULTIPLIER * TURNOVER_REDUCTION_PCT;
const reducedPresenteeismCost = headcount * PRESENTEEISM_COST_PER_PERSON * 0.10;
const totalSavings = reducedAbsenteeismCost + reducedTurnoverCost + reducedPresenteeismCost;
const platformCost = headcount * 10 * 12; // $10/seat/month
const roi = ((totalSavings - platformCost) / platformCost) * 100;
```
Display: large number cards showing totalSavings, roi (as "X% ROI"), and breakdown by category.
Inputs: sliders or number inputs for headcount (10-5000) and avgAnnualSalary ($30k-$200k).
Note below calculator: "Conservative estimates based on NIOSH/RAND/SHRM benchmarks. Actual results vary."
