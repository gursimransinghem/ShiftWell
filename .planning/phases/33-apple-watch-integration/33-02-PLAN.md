**app.json background mode configuration:**

Add HealthKit background delivery entitlement to enable overnight HRV data collection without requiring app to be in foreground.

In app.json, under `expo.ios.infoPlist`:
```json
{
  "UIBackgroundModes": ["fetch", "remote-notification"],
  "NSHealthShareUsageDescription": "ShiftWell reads your heart rate variability to improve recovery score accuracy.",
  "NSHealthUpdateUsageDescription": "ShiftWell does not write health data."
}
```

Under `expo.ios.entitlements`:
```json
{
  "com.apple.developer.healthkit": true,
  "com.apple.developer.healthkit.background-delivery": true
}
```

The `healthkit.background-delivery` entitlement allows HealthKit to wake the app when new HRV samples arrive overnight, even without UIBackgroundModes: healthkit (Apple removed that requirement in watchOS 8+).

**Watch complication documentation:**
