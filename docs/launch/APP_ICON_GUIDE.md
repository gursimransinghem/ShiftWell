# ShiftWell App Icon Guide

> Comprehensive guide to designing, generating, and configuring the ShiftWell app icon.

---

## 1. Required Sizes

### iOS / Apple App Store

| Size | Use | Required? |
|------|-----|-----------|
| **1024x1024** | App Store listing | Yes (mandatory) |
| **180x180** | iPhone @3x (60pt) | Auto-generated by Expo/EAS |
| **120x120** | iPhone @2x (60pt) | Auto-generated by Expo/EAS |
| **167x167** | iPad Pro @2x (83.5pt) | Auto-generated by Expo/EAS |
| **152x152** | iPad @2x (76pt) | Auto-generated by Expo/EAS |
| **87x87** | Spotlight @3x (29pt) | Auto-generated by Expo/EAS |
| **80x80** | Spotlight @2x (40pt) | Auto-generated by Expo/EAS |
| **76x76** | iPad @1x (76pt) | Auto-generated by Expo/EAS |
| **58x58** | Settings @2x (29pt) | Auto-generated by Expo/EAS |
| **40x40** | Notification @2x (20pt) | Auto-generated by Expo/EAS |

**Bottom line:** You only need to create a single **1024x1024 PNG**. Expo/EAS Build automatically generates every other size when building for iOS. Save it to `assets/images/icon.png`.

### Android (Adaptive Icon)

| Asset | Size | Notes |
|-------|------|-------|
| Foreground layer | 432x432 px | Transparent background; keep main graphic in center 288x288 safe zone |
| Background layer | 432x432 px | Solid color or image |

Save to `assets/images/android-icon-foreground.png` and `assets/images/android-icon-background.png`.

## 2. Design Direction

### Apple's Icon Guidelines (Must Follow)

- **No transparency / no alpha channel** — the icon must be fully opaque
- **No rounded corners** — iOS applies the superellipse mask automatically; deliver a square image
- **Avoid text in the icon** — text becomes illegible at small sizes and Apple may reject it
- **No photos or screenshots** — use graphic/illustrative style
- **Single focal point** — the icon should be recognizable at 29x29 pt
- **Avoid UI elements** — no toggle switches, app-store badges, or hardware depictions

### Color Palette

Use the ShiftWell app theme colors:

| Color | Hex | Use |
|-------|-----|-----|
| Dark navy background | `#0A0E1A` | Icon background |
| Primary purple | `#7C5CFC` | Main icon element / accent |
| Purple glow | `#9B7DFF` | Subtle glow or gradient highlight |
| Sleep purple | `#B388FF` | Secondary accent |
| White | `#FFFFFF` | Optional subtle detail |

### Concept Options

#### Option A: Crescent Moon + Clock Elements
A refined crescent moon shape in purple (`#7C5CFC`) with subtle clock-hand or hour-marker accents suggesting shift timing. The crescent sits slightly off-center on the dark navy background (`#0A0E1A`) with a soft purple glow radiating behind it. This concept immediately communicates "night" and "time/scheduling" — the two core ideas of the app.

**Strengths:** Instantly recognizable at any size. Clear association with night shifts and sleep. Differentiated from generic moon icons by the clock element.

#### Option B: Circadian Waveform
An elegant sine wave or circadian rhythm curve rendered as a glowing purple line that flows across the icon. The wave peaks and troughs suggest the rise and fall of alertness across a shift cycle. The line could subtly transition from deep purple at the "night" trough to a lighter lavender at the "day" peak, all on the dark navy background.

**Strengths:** Scientific and modern. Appeals to the evidence-based positioning. Unique — no competitor uses a waveform icon.

#### Option C: Abstract Night Gradient
A bold, abstract gradient that transitions from deep navy at the bottom to rich purple (`#7C5CFC`) at the top, suggesting the horizon at shift change. A single geometric element — a small stylized star, dot, or subtle "N" letterform — sits in the upper portion as a focal point. Minimalist and premium-feeling.

**Strengths:** Extremely clean at small sizes. Feels premium and modern. Ages well (no trendy illustrations to look dated).

**Recommendation:** Start with **Option A** (crescent + clock). It communicates the app's purpose most clearly in the App Store, where users browse quickly. The moon/night association is immediate, and the clock element differentiates it from generic sleep apps.

### Post-Generation Refinement

After generating concepts:
1. Pick the best result and download at highest resolution
2. Open in **Figma** (free) or **Canva** to:
   - Adjust background to exact `#0A0E1A`
   - Ensure purple tones match `#7C5CFC`
   - Remove any text the AI may have added
   - Verify the design reads well when scaled down to 60x60 px
3. Export as **1024x1024 PNG, no transparency, sRGB color space**

## 5. Expo Configuration

The icon is already configured in `app.json`:

```json
{
  "expo": {
    "icon": "./assets/images/icon.png"
  }
}
```

When you run `eas build --platform ios`, Expo automatically:
- Reads the 1024x1024 source from `assets/images/icon.png`
- Generates all required iOS icon sizes
- Embeds them in the app bundle
- Applies the iOS rounded-corner mask

For the Android adaptive icon, add to `app.json` under `expo.android`:

```json
{
  "android": {
    "adaptiveIcon": {
      "foregroundImage": "./assets/images/android-icon-foreground.png",
      "backgroundImage": "./assets/images/android-icon-background.png"
    }
  }
}
```
