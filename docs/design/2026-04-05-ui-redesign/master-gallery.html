Final Expert Audit — 8 Visual Improvements

Tabular Numerals Must-have

All countdown numbers use font-variant-numeric: tabular-nums so digits don't jump when counting down. Without this, "2:14" shifting to "2:13" causes the layout to jitter.

Noise Texture Overlay Polish

2.5% opacity fractal noise over the background. Adds depth and tactile quality. Hallmark of Linear, Raycast, Arc. Already applied in this gallery.

Warmer Background Polish

Shifted from cold #0A0E1A to warmer #0B0D16. Subtle but makes the app feel cozy rather than clinical. Critical for a sleep app.

Consistent 8px Vertical Rhythm Must-have

All margins between sections snap to 8px multiples (8, 16, 24). Prevents the "something feels off" sensation from inconsistent spacing.

Card Elevation Hierarchy Nice-to-have

Three card levels: flat (timeline), raised (countdowns, insight), floating (tab bar). Each gets progressively more border opacity + shadow. Creates visual depth.

Active State Gradient Direction Polish

Active timeline cards have a left-to-right gradient (color fades from left). This draws the eye to the event name first, then naturally to the countdown on the right.

Score Ring Accuracy Nice-to-have

The arc should be mathematically proportional to the score (78% = 281 degrees). Use SVG with stroke-dasharray for pixel-perfect arcs.

SF Symbols for Tab Icons Must-have

Replace emoji tab icons with SF Symbols (sparkles, calendar, person.circle) for native iOS feel. Emoji in tabs looks amateur on iOS.
