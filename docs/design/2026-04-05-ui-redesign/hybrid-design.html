Component Origins & Rationale

Header AB

Path A's personalized greeting + large date. Path B's gradient avatar. Contextual subtitle adapts to state.

Status Pill B

Contextual banner that changes color + icon per state: green (recovery), orange (on shift), indigo (wind-down).

Animation: subtle glow pulse every 3s

Recovery Score A

Larger inline card with ring, trend arrow, and natural language insight. More prominent than B's compact bento version.

Animation: ring glow pulse every 4s

Bento Cards B

Color-coded gradient cards for key countdowns. Large numbers for glanceability. Subtle press-down scale on tap.

AI Insight B

Gold-bordered card with contextual, forward-looking sleep advice. Changes based on shift pattern and sleep history.

Animation: fade-in slide-up on appear (0.6s)

Timeline AB

Path A's spine with dots (visual progression) + Path B's inline countdown labels (Done/Now/5h 30m). Best of both.

Animation: active dot breathing glow (1.5s loop)

Floating Nav B

Glassmorphic tab bar with blur backdrop. Renamed: Today / Schedule / Profile. Active icon has gold glow.

Night Sky Enhanced

Beyond current: firefly particles, glowing arc with ambient pulse, varied star twinkle speeds, structured tomorrow card.

Animation: 4 fireflies + arc glow pulse + 16 stars at 3 speeds + slow arc rotation
