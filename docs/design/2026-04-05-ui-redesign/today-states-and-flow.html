5. Night Sky (Sleeping)

Navigation Flow

How users move through the app in both paths

Path A: 3 Tabs (Current)

  • Today → Timeline, countdowns, recovery, insight, Night Sky overlay
  • Schedule → Month calendar, day detail, add/edit shift
  • Settings → Account, sync, notifications, profile, import/export, danger zone
  • Night Sky is a full-screen overlay triggered by bedtime
  • Onboarding is a separate stack (8 screens, linear)

Path B: 3 Tabs (Renamed)

  • Today → Bento dashboard, contextual status, timeline, insights
  • Schedule → Week strip + month toggle, icon shift cards, add FAB
  • Profile → Avatar, stats, preferences, data, account (replaces Settings)
  • Night Sky transitions smoothly from wind-down state (no hard overlay)
  • Onboarding is swipeable cards (dot navigation, 4-5 screens, skippable)
