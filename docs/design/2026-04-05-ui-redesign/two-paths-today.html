Path A Changes

  • Typography: Personalized greeting + contextual subtitle added
  • Recovery score: Moved inline with trend + insight text
  • Countdown cards: Subtle gradient on primary card, tighter spacing
  • Timeline: Thinner spine, gradient highlights on active/next
  • Emoji icons: Added inline for scan-ability
  • Spacing: Tightened throughout for better information density
  • Structure: Identical to current — 3 tabs, scrollable timeline
  • Effort: ~1-2 days of styling work

Path B Changes

  • Layout: Bento grid dashboard replaces horizontal countdown scroll
  • Status pill: Contextual banner (recovery day, on shift, etc.)
  • Recovery: Compact inline card within bento grid
  • Timeline: Flat list with inline countdowns (no spine/dots)
  • Insight card: AI-powered contextual insight elevated to main view
  • Nav: Floating glassmorphic tab bar + avatar in header
  • Background: Deeper black (#080B14) + glass-border cards
  • Effort: ~3-5 days, more structural component changes
