To keep this dashboard current automatically, add a git post-commit hook:

```bash

#!/bin/bash
bash "$(git rev-parse --show-toplevel)/scripts/update-progress.sh"
git add docs/PROGRESS-DASHBOARD.md 2>/dev/null
git commit --amend --no-edit --no-verify 2>/dev/null || true
```

```bash

```
